# L-tyrosine
Data for "L-tyrosine protects cognition in a preclinical model of synucleinopathy"

Files: 
- LtyrosineCognitive.xlsx : DRT scores for marmosets
- LtyrosineGait.xlsx : 6 different gait assessments, with each gait parameter as a different tab in the spreadsheet
- LtyrosineASYNandTau : Measurements of ASYN and Tau by region for each marmoset
- LtyrosineASYNnormalized : ASYN data after normalizing to the control group; used to fit the repeated measures (mixed) model to the normalized ASYN in each of 5 regions per marmoset
